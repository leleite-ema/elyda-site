# Élyda Maele — Nutrição Online (Next.js + Tailwind)

## 1) Pré-requisitos
- Node.js 18+ instalado

## 2) Rodar local
```bash
npm install
npm run dev
```

## 3) Ajustes rápidos
- WhatsApp: `components/site-config.ts` (campo `whatsappNumber`)
- SEO: `components/site-config.ts` (title/description/url)

## 4) Publicar na Vercel (grátis)
- Suba este projeto no GitHub
- Vercel -> New Project -> Import Git Repository -> Deploy
